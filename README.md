## 💡 项目愿景

TradeInsight AI 是一个**专为交易者打造的智能知识助手**，它能够：

- 📚 **理解并记忆**你上传的所有交易教程、策略文档、市场分析

- 🎯 **精准回答**关于 ICT、SMC 等交易方法论的复杂问题

- 🔍 **语义检索**相关知识片段，而非简单的关键词匹配

- 💬 **自然对话**像与交易导师交流一样获取知识

**核心价值**： 将散落在视频、PDF、网页中的交易知识，转化为可随时调用的个人智能顾问。

## 🏗️ 技术架构

```plaintext
┌─────────────────────────────────────────────────────────┐
│                    用户交互层                              │
│  ┌──────────────┐         ┌──────────────┐              │
│  │  Web界面     │         │  API接口     │              │
│  │  (Streamlit) │         │  (FastAPI)   │              │
│  └──────────────┘         └──────────────┘              │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│                    核心RAG引擎                            │
│  ┌──────────────────────────────────────────────────┐   │
│  │  1. 文档处理    →  2. 向量化    →  3. 存储      │   │
│  │  (PDF/TXT/MD)      (Embedding)     (ChromaDB)   │   │
│  └──────────────────────────────────────────────────┘   │
│  ┌──────────────────────────────────────────────────┐   │
│  │  4. 语义检索    →  5. 上下文构建  →  6. 生成    │   │
│  │  (Vector Search)   (Prompt)        (DeepSeek)   │   │
│  └──────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│                    数据存储层                             │
│  ┌──────────────┐         ┌──────────────┐              │
│  │  向量数据库   │         │  文档存储     │              │
│  │  (ChromaDB)  │         │  (本地/云端)  │              │
│  └──────────────┘         └──────────────┘              │
└─────────────────────────────────────────────────────────┘
```

### 技术栈

| 层级 | 技术选型 | 理由 |
| --- | --- | --- |
| **LLM** | DeepSeek V3 | 性价比最高的中文大模型，推理能力强 |
| **向量数据库** | ChromaDB | 轻量级、易部署、支持本地运行 |
| **Embedding** | ChromaDB 默认模型 | 开箱即用，无需额外配置 |
| **Web 框架** | Streamlit | 快速构建数据应用，专注逻辑而非前端 |
| **API 框架** | FastAPI | 高性能异步框架，便于后续扩展 |

## 🚀 快速开始

### 前置要求

- Python 3.8+

- DeepSeek API Key （[获取地址](https://platform.deepseek.com/)）

### 安装步骤

```bash
# 1. 克隆项目
git clone https://github.com/Jade-ops-tech/tradeinsight-ai.git
cd tradeinsight-ai

# 2. 创建虚拟环境
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置API密钥
cp .env.example .env
# 编辑 .env 文件,填入你的 DEEPSEEK_API_KEY
```

### 运行示例

**当前版本 （命令行）**:

```bash
python minimal_rag.py
```

**未来版本 (Web 界面）**:

```bash
streamlit run app.py
```

## 🎯 项目亮点

### 1️⃣ **第一性原理设计**

不是简单堆砌技术，而是从“为什么 LLM 需要外部知识”这个本质问题出发，构建最小化但完整的 RAG 系统。

### 2️⃣ **工程化思维**

- 虚拟环境隔离依赖

- 环境变量管理敏感信息

- 模块化代码结构

- 完整的 Git 工作流

### 3️⃣ **成本优化**

使用 DeepSeek(¥1/百万 tokens）替代 GPT-4(¥210/百万 tokens)，在保证效果的同时降低 99% 成本。

### 4️⃣ **可扩展架构**

从 50 行 MVP 到完整产品，每个版本都是可运行的，支持渐进式开发。
